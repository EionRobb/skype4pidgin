Install
=======
Close Pidgin
Run pidgin-skypeweb.exe

Configuration
=============
- Start Pidgin
- Go to menu: Accounts - Manage Accounts
- Click Add...
- Protocol: Select Skype (HTTP)
- Username: Enter your Skype id
  IMPORTANT: this is not the Microsoft Account email address
- Enter the password
- Click Add
